About This Project

This application allows users to encrypt and decrypt text files securely using a Java-based backend. It also features user feedback and login systems, integrated with MongoDB Compass to store and manage data.

The goal of this tool is to provide a secure, user-friendly interface for file encryption while collecting feedback and login info to enhance functionality and security in the future.
